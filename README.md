# PRODIGY_GA_04

## Image-to-Image Translation with cGAN (pix2pix)

### 📌 Task
Implement an image-to-image translation model using 
a conditional generative adversarial network (cGAN) 
called pix2pix.

### 🛠️ Tools & Libraries Used
- Python
- PyTorch
- torchvision
- Matplotlib
- Google Colab (T4 GPU)

### 🏗️ Architecture
- Generator: U-Net architecture with skip connections
- Discriminator: PatchGAN architecture
- Loss: BCE Loss + L1 Loss

### 🔗 What I Built
- U-Net Generator from scratch
- PatchGAN Discriminator from scratch
- Training loop with adversarial loss
- Visualization of input, output & difference map
- Training loss graph
